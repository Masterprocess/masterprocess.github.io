Vinum Vitae

Check it out here!

This was a group project that I worked on with Cory Lozon and Adrian Fahrer. I primarily worked the backend while Cory worked out the API, and Adrian built up the front end. Our idea was pretty simple. We just wanted an easy way to find new wines. What we ended up building used a wine api to search for wines, a database to store user wines, and a user interface to interact with it all. The site was built in only two weeks, so it's a little rough around the edges, but it does what we wanted it to and that was a qualified success in our book. Eventually we ended up having to remove the hosting of this project as we didn't feel a need to pay for server maintenance. You can still see the code that went behind its creation on GitHub, though. Perhaps in the future, I'll revisit the project and add the optical character recognition tool that I never had time to implement.

  • Date: September 2017
  • Client: Vinum Vitae
  • Category: Wines

Reboot Camp Spot

Check it out here!

This was my final project in the UNC Coding Bootcamp. I worked on this project with Adrian Fahrer who built the frontend, Li Chen who helped to architect the backend with me, and Caroline Bowes who helped sooth the gremlins in Angular4. In retrospect, we bit off more than we could chew. We still shipped a minimum viable product, but many features that were initially planned had to be scrapped due to strict time constraints. This was our first attempt at building a full stack app in Angular. The goal was to create an app that could facilitate classes at the bootcamp by allowing TAs to grade homework and mark attendance. We were ultimately successful in achieving this. If I had the time, I would go back and change the app to use MySQL instead of MongoDB so as to more effectively facilitate homework tracking. Overall, I'm proud of what we built with this.

  • Date: November 2017
  • Client: UNC Coding Bootcamp
  • Category: Blackboard Knock-Off
